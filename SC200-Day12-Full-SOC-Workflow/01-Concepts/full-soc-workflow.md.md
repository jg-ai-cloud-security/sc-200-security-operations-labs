\# Microsoft Defender XDR Threat Hunting



Microsoft Defender XDR provides a unified security operations experience that helps analysts investigate threats across multiple domains such as endpoints, identities, email, and cloud applications.



Threat hunting in Defender XDR is a proactive SOC activity where analysts search security telemetry for suspicious behavior that may not yet have triggered a high-confidence incident.



A key capability used in threat hunting is Advanced Hunting, which allows analysts to query raw security data using Kusto Query Language (KQL).



\## Key Concepts

\- Microsoft Defender XDR

\- Advanced Hunting

\- Kusto Query Language (KQL)

\- Threat analytics

\- Endpoint, identity, and email telemetry

\- Proactive investigation

\- Detection engineering



\## Why It Matters

Threat hunting helps a SOC move beyond alert monitoring into proactive discovery. It enables analysts to identify suspicious activity, validate attack hypotheses, and improve detection coverage.
