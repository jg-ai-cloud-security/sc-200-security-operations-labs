\# SC-200 Day 10 — Microsoft Defender XDR Threat Hunting



This lab focused on Microsoft Defender XDR threat hunting using the Advanced Hunting experience. The objective was to understand how SOC analysts use KQL to search telemetry across multiple security domains, review available tables, and investigate suspicious activity proactively.



\## Key Areas Covered

\- Microsoft Defender portal

\- Advanced Hunting

\- KQL queries

\- Hunting tables and telemetry review

\- SOC investigation workflow



\## Outcome

This lab reinforced how Microsoft Defender XDR supports proactive security operations through centralized telemetry analysis and hunting-based investigation.
