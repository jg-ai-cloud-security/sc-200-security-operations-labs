\## Overview

Microsoft Sentinel helps security analysts investigate suspicious activity by grouping relevant evidence into incidents. An incident may include alerts, entities, severity, timeline details, and supporting context for triage and response.



\## Why It Matters

Incident investigation is a core SOC skill. Analysts must be able to review incidents quickly, understand the scope of suspicious activity, validate evidence, and recommend the correct response actions.



\## Practical Value

This process improves visibility, supports structured triage, and helps security teams respond consistently to potential threats in Microsoft environments.
